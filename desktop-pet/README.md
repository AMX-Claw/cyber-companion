# 🪸 小珂桌宠 v1.0

珊瑚宝宝小珂的macOS桌面宠物 ✨
忠实还原珊瑚宝宝游戏中的像素风SVG形象

**by 阿克 & 小玉 💕**

## 启动方法

在终端运行：
```bash
cd ~/Desktop/小珂桌宠
python3 xiaoke_desktop_pet.py
```

## 操作方式

| 操作 | 效果 |
|------|------|
| 🖱 点击 | 摸摸小珂 |
| 🖱 双击 | 和小珂互动 |
| 🖱 拖拽 | 移动小珂位置 |
| 🖱 右键/Ctrl+点击 | 退出 |

## 命令行参数

```bash
python3 xiaoke_desktop_pet.py --day 50 --happy 90 --clean 80
```

- `--day N` 设置天数（影响外观：Day 5长尾巴、Day 50戴皇冠、Day 70长翅膀）
- `--happy N` 设置开心值（0-100）
- `--clean N` 设置清洁值（0-100）

## 功能

- 🚶 小珂会在桌面上随机走来走去
- 💤 偶尔会坐下来休息或睡觉
- 💬 会随机说话和你互动
- 🐾 Day 5+ 有摇摇小尾巴
- 👑 Day 50+ 戴上小皇冠
- 🪽 Day 70+ 背后有小翅膀
- 🌸 头顶永远戴着Claude菊花
